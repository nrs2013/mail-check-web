# Mail-Check! Web版（GitHub Pages版）

Gmail を使ったメール仕分けダッシュボード（独立Webアプリ）。
GitHub Pages で完全無料ホスティング。

## 本番URL

https://nrs2013.github.io/mail-check-web/

## 機能

- Google でログイン（Gmail OAuth・ブラウザ直接）
- 自分宛・自社・To/Cc メール仕分け
- AI要約（Claude Haiku 4.5・有料）
- PIC UP! 案件サマリー

## 必要なもの

- **Claude API キー**（Anthropic console で発行）
  - 課金制：使った分だけ。Haiku 4.5 は安い（月数百円〜千円）
